Free portfolio and image gallery plugin for WordPress, customized by DevsAir (https://devsair.com).


== Description ==

**DevsAir Portfolio Elementor** is a free, customized portfolio plugin maintained by [DevsAir](https://devsair.com). Build filterable portfolio grids and image galleries with Elementor widgets, Gutenberg blocks, or shortcodes.

**About DevsAir**

DevsAir provides WordPress development and customization services. Learn more at **https://devsair.com**.

**Main features**

* Portfolio custom post type with categories
* Elementor portfolio and image gallery widgets
* Gutenberg portfolio and image gallery blocks
* Shortcode: `[powerfolio]`
* Masonry and boxed grid layouts
* Category filters and hover effects
* Lightbox or single project page links

Set a **Featured image** on each portfolio item so it appears in the grid.


== Installation ==

1. Upload the `portfolio-elementor` folder to `/wp-content/plugins/`.
2. Activate the plugin in **Plugins**.
3. Open **DevsAir Portfolio** in the admin menu for setup notes.
4. Add portfolio items under **Portfolio** and set a featured image on each item.
5. Add the Elementor widget or Gutenberg block to a page, or use `[powerfolio]`.


== Frequently Asked Questions ==

= Why is my portfolio image missing? =

Edit the portfolio item and set the WordPress **Featured image** (not only an image in the post content).
